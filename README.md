# ddroidd_ Frontend Winter Internship 2022 Web Challenge

## Run the following commands in the terminal:

### node src/server.js

It starts the server on port 3000, the server is only used for the form post request and it logs the data in the console.

### npm start

It starts the main app on port 8080.
